jQuery

  • JQuery -> Javascript Query -> Consulta Javascript
  • É uma biblioteca JavaScript extremamente popular
  • Ajuda na busca de elementos e alteração da DOM
  • Trabalha com manipulação de eventos, animações, Ajax...
  • Pensado para trabalhar com conjunto de elementos
  • Usa seletores CSS
  • Encadeamento de chamadas (chaining)
  • É de fácil extensão usando plugins
  • Suporte múltiplos Browsers
  • Só tem pontos positivos? Não!
    • A direção atual com frameworks e afins é a de criar componentes reutilizavéis, (que se liga a POO) o que não necessariamente é o caminho do Jquery (mais procedural)
      • O jQuery altera diretamente a DOM com várias funções utilitárias/plugins para auxiliar no mesmo
      • Alguns benefícios de performance que temos com os frameworks não existem com o jQuery
      • O mesmo tem seus usos e auxílios, mas não é ""sustentável"" para uma aplicação do zero coms os métodos atuais (isso pensando em sistemas grandes, principalmente)
      • Ainda tem bastante uso, principalmente com templates e com o próprio Bootstrap, que usa o jQuery
